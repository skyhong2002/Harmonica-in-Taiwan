🌟 本次2026臺灣口琴音樂節，將匯聚來自亞洲各國的口琴演奏家同台演奏！此次邀請的一位重量級嘉賓是來自日本，享譽國際的半音階口琴演奏家 @minami_risa 南 里沙 ! 💪
南里沙，半音階口琴演奏家，畢業於神戶女學院大學音樂學系（主修雙簧管）。在接觸半音階口琴後深受其魅力吸引，開始投入學習，並一路持續至今，發展為其終身熱愛的演奏志業。
2009年，於德國 F.I.H. 國際口琴比賽公開組獲得第二名；其後於第八屆亞太口琴節（新加坡）半音階獨奏公開組榮獲第一名。
南里沙亦為全球首位與保加利亞國家索菲亞愛樂管弦樂團合作，完整演出《Labyrinth of Fantasy》（半音階口琴協奏曲）四個樂章的演奏家，演出獲得熱烈迴響。
2013年，以首張專輯《Mint Tea》由 King Records 正式出道。近年演出與錄製工作廣泛，包括參與 NHK 電視節目《Anthology Album》，並與東京愛樂交響樂團、東京混聲合唱團合作演出，同時亦參與多部戲劇與動畫配樂錄製。
憑藉精湛且高水準的半音階口琴演奏技巧，以及跨越音樂領域的多元活動，南里沙在日本國內外累積了眾多樂迷與音樂專業人士的支持與喜愛。現為 HOHNER Artist。

Harmonica Warm Up nr 2 (recorded live video, unedited)
Harmonica warm up #2!
This harmonica warm up features a single note scale motion in a repetitive pattern. Doing this really opens up the breathing, single note stability and pitch control. Do it maximum for 1 minute sessions, maybe 3-5 sessions as a warm up. Take it easy and slow first - work up the speed and don't rush it.
We are using the first five notes of the major scale, hole 4-6 on diatonic and holes 5-7 on chromatic harmonica. You can do either the eight note version or 16th note version.

某天晚上，口琴社燈火未熄

某天晚上，口琴社燈火未熄
幹部們正圍坐在社辦裡，思考著社團未來的興衰存亡
此時，一道黑影緩緩走來
牠的腳步很輕
但壓迫感很重
是黑黑團長。
黑黑跳上桌面，環視眾人，尾巴輕輕一甩，氣氛瞬間安靜
幹部們低頭不語，沒有人敢先開口
畢竟大家都知道，當黑黑團長主動現身，通常不會有什麼好事
「團長……這麼晚了，您有何吩咐？」
黑黑瞇起眼睛，看向眾人
「喵。」
社辦瞬間陷入死寂
「團長的意思是……？」
黑黑再次開口
「喵喵喵喵🔥」
眾幹部大驚失色
「什麼？！團長要我們再招募更多鏟屎官來服侍您？！」
「團長，別為難臣妾呀！」
「如今各個社團多彩多姿，學生選擇眾多，要把社員留下來已是不易，更何況還要找人一起扛社團大業！」
「現在大家課業繁忙、活動眾多、訊息已讀不回、幽靈人口四處飄散，臣妾真的做不到啊！」
黑黑沉默了三秒
然後伸出貓掌，輕輕拍了一下桌子
「喵。」
那一聲不大，但每個人都聽懂了
意思是：不要找理由，去招募
幹部們互看一眼，眼神中充滿了恐懼、責任、疲憊，以及一點點被貓咪支配的幸福
於是，這份招募文誕生了。
—
想來一個可以玩貓咪的社團嗎？
很好，歡迎來口琴社，黑黑團長正在募集鏟屎官
當你看到這裡的時候，代表你已經得到黑黑的注視
而且這不是普通的注視，這是來自團長的無聲審核、精神召喚，以及「不加入就要把你阿罵賣掉」的暗示
加入口琴社，你可能會被黑黑選中，成為維持社團運轉的重要鏟屎官！
當然，我們不是只有缺鏟屎官，我們還有以下職缺：
副社長／活動／教學／美宣／公關／器材／總務／場務／資訊／網管／背鍋仔
只要你想得到的幹位，我們都缺！甚至你想自己取一個喜歡的職稱也可以！！！
如果有院內同仁看完後突然覺得「欸我好像也可以幫忙」，也歡迎一起加入黑黑團長的麾下
不分學生、同仁、路過被貓咪吸引的人，只要願意一起讓社團活下去，黑黑都會納入觀察名單
職位先搶先贏，事情後來再說，我們需要的人其實很簡單：
1. 會呼吸
2. 願意回訊息
3. 看到事情願意想辦法解決
4. 不要在關鍵時刻突然變成都市傳說
只要你符合以上條件，就已經贏過很多幽靈人口了
你也不需要一開始就很會吹口琴，甚至不會任何樂器也沒關係，只要你有興趣、有想法、有一點點熱忱，或是單純覺得黑黑很可愛，都有機會被黑黑團長選中，成為下一屆鏟屎官
不用怕你不會當幹部，因為沒有人天生會當幹部，我們當初也不會，只是某一天，我們以為自己只是來社團玩一下，結果突然開始開會、辦活動、搬器材、發公告、催人回訊息、思考這首歌到底誰要吹
然後就回不去了。
但你問我們後悔嗎？
可能有一點，但不多
因為吸貓一時爽，一直吸貓一直爽
而且在這裡，你會遇到一群一起努力、一起崩潰、一起把事情完成的人
可能會很荒謬、很忙、很累，但也會是你大學裡很難忘、很充實、很值得回頭看的回憶
所以，如果你願意一起讓口琴社繼續熱鬧下去
如果你願意成為下一屆傳說中的鏟屎官
