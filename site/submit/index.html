What To Report

可以回報什麼

只收公開可查資料

新增來源或社團

新增學校社團、樂團、演奏者、教學工作室、場館、公開社群頁或 YouTube 頻道。

需要官方或公開來源 新增來源

修正既有資料

名稱、分類、地區、公開連結或來源顯示有誤時，可以回報建議修正。

請附目前頁面或來源 修正資料

活動、比賽與補助

回報公開演出、成發、講座、工作坊、比賽、徵件、報名或補助公告。

請附日期與公告頁 新增活動資訊

移除或停止收錄

連結失效、頁面停用、來源不再公開，或資料不適合收錄時，可以回報下架原因。

請說明判斷依據 停止收錄

Before You Submit

回報前確認

Issue 會公開顯示

請提供官方網站、公開社群頁、公開貼文、活動頁、售票頁或其他公開可查來源。

請直接寫出希望網站最後呈現的結果；表單會另外詢問 agent 要驗證的頁面或 API。

不要填私人電話、私人信箱、未公開群組連結、會員資料、憑證或任何不方便公開的資料。

查看處理中的回報 開啟 GitHub 回報選單
